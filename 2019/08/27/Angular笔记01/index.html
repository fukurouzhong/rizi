单向绑定

单向绑定是说只要component中的值发生变化，自动更新UI中的元素
绑定的几种形式：
interpolation
property Binding
event Binding

这段代码中我们可以观察出几种单向绑定的方式
undefined (undefined)称做interpolation，
而[class]=”positiveChange ? ‘positive’ : ‘negative’”可以动态绑定class
而(click)可以动态绑定event

Module

To define an interface or a class with the definition for what belongs in a stock

我们仍然通过CLI命令生成一个model

ng generate class model/stock

然后在需要引入model的文件中引入
import { Stock } from '../../model/stock'

然后我们就能在引入的文件中使用

在constructor中生成的property我们通过new Stock的方式初始化。
this.stock = new Stock('Test Stock Compnay','TSC', 85, 80)

初始化之后我们就可以使用定义的property了。
